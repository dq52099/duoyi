## 优化内容

### 1. 设计系统 (DesktopTokens)

创建了统一的桌面端设计常量：

```dart
// 字体大小
fontSizePageTitle: 28px        // 页面主标题
fontSizePageSubtitle: 15px     // 页面副标题
fontSizeCardTitle: 20px        // 卡片大标题
fontSizeCardSubtitle: 16px     // 卡片小标题
fontSizeBody: 15px             // 正文
fontSizeCaption: 13px          // 辅助文字

// 间距
pageHorizontalPadding: 40px    // 页面左右外边距
pageTopPadding: 32px           // 页面顶部外边距
cardSpacing: 20px              // 卡片间距
columnSpacing: 28px            // 列间距

// 布局尺寸
maxContentWidth: 1280px        // 最大内容宽度
leftSidebarWidth: 420px        // 左侧边栏宽度
rightSidebarWidth: 380px       // 右侧边栏宽度

// 断点
breakpointThreeColumn: 1400px  // 三栏布局最小宽度
breakpointTwoColumn: 1100px    // 两栏布局最小宽度
```

### 2. Header 优化

**优化前**:
- 标题: 22px, normal weight
- 日期: bodySmall (约 12px)
- 无背景卡片

**优化后**:
- 标题: 28px, w600 (semibold)
- 日期: 15px
- 带卡片背景和边框
- 内边距: 20px
- 更高对比度

### 3. 三栏布局

#### 左侧边栏 (420px, 宽度 ≥ 1400px 显示)

1. **精简月历**
   - 当月日历网格
   - 高亮今日
   - 周末日期变灰

2. **今日一言**
   - 从主内容区移到左侧

3. **快捷入口面板**
   - 习惯打卡
   - 番茄专注
   - 快速笔记
   - 日程安排
   - 2x2 网格布局

#### 主内容区 (Flex)

- 黄历 + 效率评分（响应式行）
- 指标网格（4列）
- 待办列表
- 提醒列表

#### 右侧边栏 (380px)

- 目标进度
- 课程安排
- 纪念日提醒

### 4. 指标卡片优化

**字体大小**:
- 标题: 11px → 13px
- 数值: 12.5px → 15px (w500)
- 单位: 11px → 13px

**尺寸**:
- 内边距: 12x10 → 14x12
- 图标盒子: 28px → 32px
- 图标: 15px → 17px

### 5. 对比度提升

**文字透明度**:
- 主要文字: 默认 → 0.95
- 次要文字: 默认 → 0.87
- 辅助文字: 0.62 → 0.70

**卡片样式**:
- 背景透明度 (亮色): 0.78 → 0.85
- 背景透明度 (暗色): 0.62 → 0.68
- 边框透明度 (亮色): 0.12 → 0.18
- 边框透明度 (暗色): 0.10 → 0.15
- 边框宽度: 0.45px → 0.5px

### 6. 响应式断点

```
宽度 ≥ 1400px:
  [左侧边栏 420px] [主内容 flex] [右侧边栏 380px]
  
1100px ≤ 宽度 < 1400px:
  [主内容 flex] [右侧边栏 380px]
  (今日一言移到主内容区顶部)
  
宽度 < 1100px:
  切换到移动端布局
```

## 部署验证

```bash

  --base-href=/duoyi/ \
  --dart-define=DUOYI_SERVER_URL= \
  --dart-define=DUOYI_WEB_TARGET=desktop
```
