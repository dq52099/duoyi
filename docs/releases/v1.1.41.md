# v1.1.41

本版本修复提醒设置、待办提交和今日任务统计问题，并加固 Android 软件外全屏闹钟弹出链路。

## 修复

- 通知设置页不再平铺展示所有注册提醒，改为进入“已注册提醒明细”二级页查看。
- 已注册提醒明细改用任务标题、提醒类型和系统记录数量展示，不再把对象 ID 或队列 ID 串直接暴露给用户。
- 待办创建和保存成功后不再弹出“旧提醒清理失败”等非阻断内部错误；提醒保存失败仍保留真正阻断的前置校验。
- 待办今日进展统一使用未完成口径，避免“今日还要完成 2 项 / 日常 1 / 待办 2 / 目标 1”这类总数和细项不一致。
- Android 原生全屏闹钟服务通知补齐系统 full-screen intent，减少应用在后台或锁屏时只响铃、不弹出全屏界面的情况。
