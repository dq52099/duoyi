# v1.1.22

本次发布继续收口万年历黄历详情页、全局文字层级、通知设置安全区、主题化万年历卡片和内置提醒铃声试听。

## 重点修复

- 黄历详情页改为稳定纵向布局，十二时辰不再覆盖中部信息，彭祖和冲煞完整显示。
- 宜忌区域改为更紧凑的竖排展示，并保留统一的查看全部入口，完整黄历数据源不变。
- 全局文字层级继续收敛，二级页面、按钮、设置项和底部导航不再回到粗大标题风格。
- 通知设置页修复滚动内容进入顶部标题栏和安全区的问题，卡片边框、间距和展开态层级更克制。
- 首页万年历卡片和桌面小组件主题色改为跟随当前主题背景，不再固定淡蓝默认块。
- 内置提醒铃声试听改为前台直接播放当前内置音频，不再依赖通知渠道或通知权限触发；切换铃声和音量会立即应用并试听。
- 铃声预览失败时区分资源缺失、系统闹钟音量为 0、播放器初始化失败和平台通道异常。
- 离开通知设置页或停止强提醒时释放正在试听的播放器。
- 保持 release 签名链路不变，继续使用仓库 secrets 中的正式 keystore，不改签名配置。
