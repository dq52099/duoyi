# v1.1.37

本版本修复新建目标页窄屏展示问题，并加固提醒编辑器在小屏幕下的布局稳定性。

## 修复

- 新建目标页的“开始日期 / 目标日期”字段在窄屏下改为上下排列，避免目标日期文字被边框裁切。
- 目标日期文案从“目标”明确为“目标日期”，减少字段含义不清。
- 目标提醒区域的“开启提醒”开关改为响应式布局，标题和摘要不再被开关挤出。
- 提醒方式四段选择器增加横向滚动兜底，避免小屏或较大字体下溢出。
