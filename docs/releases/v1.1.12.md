# v1.1.12

本次发布集中修复高刷设备卡顿发热、登录注册与我的页视觉、左滑详情/删除交互，以及旧后端接口诊断文案外露。

## 重点修复

- 番茄钟计时刷新收敛到局部组件，避免秒级计时触发整页重建。
- 番茄钟自由计时圆环改为静态进度，避免 120Hz 下持续无意义动画。
- 待办临期图标取消循环闪烁动画，减少列表多实例 ticker。
- 全局品牌背景增加重绘隔离，番茄钟在主壳内不再叠加双层背景。
- 登录/注册页面改为单层紧凑卡片，降低标题、提示和按钮的视觉压迫。
- 我的页用户信息区改为紧凑横向布局；头像点击全屏预览，右下角笔按钮用于编辑头像，资料区点击进入个人资料。
- 待办、习惯、今日、反馈列表左滑操作收窄为轻量图标按钮，详情/删除不再大面积挤压正文。
- 共享、评论、更新检查等接口异常统一转换为用户可读文案，避免普通页面直接显示后端部署诊断。
