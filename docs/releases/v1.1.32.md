# v1.1.32

本版本修复账号切换隔离、提醒多通道注册、通知历史未读、小组件交互和近期 UI 回归。Android `versionCode` 提升到 `130104`；签名配置保持不变。

## 修复与优化

- 退出登录和账号切换前清理本机账号数据，避免 admin 的时光币、主题、习惯、头像、小组件数据串到 test 账号。
- 修复通知、弹出框、闹钟、全屏闹钟和定时诊断提醒的注册、兜底和可见诊断链路。
- 通知历史默认保持未读，需要手动单条或批量标记已读。
- 优化黄历 6 月 28 日宜忌溢出、AI 回顾折叠、纪念日/生日摘要和小组件预览。
- 待办/月历小组件支持 2x2、3x2、4x3，待办未完成标记为 `o`，并补充快速添加、打开详情、完成任务和今天按钮等交互。

## 发布说明

- Android `versionName`: `1.1.32`
- Android `versionCode`: `130104`
- 签名配置保持不变，正式 Release 仍由 GitHub Actions 使用既有 release keystore 构建。
