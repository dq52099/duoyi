# v1.1.16

本次发布继续收口近期反馈中的 P0 功能、日历体验和发布前回归问题。

## 重点修复

- 月日历改为中等高度：4/5/6 行月份分别限制在 240-280、268-306、296-332，避免过高挤压详情，也避免过矮难点选。
- 月视图保留整页滚动，月格与当天详情一起滚动，不再只有详情区域滚动。
- 倒数日保持在“我的 / 日程日期”入口，不再出现在“更多应用”列表。
- 今日待办和今日提醒使用与新建任务一致的语义图标布局，弱化状态按钮对图标的干扰。
- 修复后台接口契约、用户/用户组、时光币、邮箱、专注自习室相关回归风险，并补充对应测试。
- Android 明确允许当前后端域名的 HTTP 访问，避免服务可用但客户端被网络安全策略拦截。
- 保持 release 签名链路不变，继续使用仓库 secrets 中的正式 keystore。
