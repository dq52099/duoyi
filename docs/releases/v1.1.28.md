# v1.1.28

本版本重点修复提醒铃声试听、桌面小组件主题背景、启动与滚动卡顿，以及通知设置、习惯页和黄历详情的可用性问题。Android `versionCode` 提升到 `130100`，避免覆盖安装被系统识别为降级。

## 修复与优化

- 铃声试听改为优先使用 Flutter asset 前台播放器，失败时再走 Android raw `MediaPlayer` 兜底；试听不再依赖通知权限、通知渠道或强提醒服务。
- 新增 28 个内置提醒铃声 asset，并同步打包到 Android `res/raw`，保证前台试听和原生兜底使用同一批音频。
- 桌面小组件保存并读取主题背景图 key，带图主题会渲染同源背景图、圆角裁切和轻量蒙层，不再只显示主题色块。
- 主题切换后主动刷新全部桌面小组件；快捷添加超时会清理 pending provider，减少添加失败后的残留状态。
- 更新后首次进入和杀后台重进时，将通知/闹钟完整初始化、提醒重同步、日历同步、小组件恢复和云同步延后执行，降低首屏阻塞。
- 通知设置页优化“提醒方式”布局，避免 label 裁切和“通知 / 弹出框 / 闹钟 / 关闭”横向拥挤。
- 习惯页压缩本周概述、热度图分组和任务摘要密度，统一更小的卡片标题、任务名和次级统计字阶。
- 黄历详情调整宜忌、彭祖、五行、冲煞的拆行和字号层级，保留动态黄历数据来源。
