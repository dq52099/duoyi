# v1.1.35

本版本补上本机数据隔离、待办滑动稳定性和 Android 闹钟定时兜底。

## 修复

- 退出账号后清理本机账号态数据，避免 admin 切到 test 后继续看到旧账号的时光币、主题、习惯等数据。
- 待办列表滚动时自动收起左滑操作区，避免“完成/删除”操作区跟随屏幕滚动悬浮。
- Android 定时闹钟和全屏闹钟注册为原生闹钟加 Flutter 本地通知双队列；原生到点触发会取消兜底，原生定时被系统拦截时仍保留备用提醒。
- 强提醒全屏测试改为真正请求全屏提醒，便于验证弹屏权限。

## 版本

- Android `versionName`: `1.1.35`
- Android `versionCode`: `140001`
