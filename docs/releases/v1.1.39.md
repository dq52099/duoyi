# v1.1.39

本版本拆分电脑 Web 和手机 Web 构建，并修复通知设置等设置行的图标、标题、副标题、右侧操作垂直对齐。

## 修复

- 电脑 Web 包使用 `DUOYI_WEB_TARGET=desktop` 独立构建，默认隐藏不支持的“小组件”底部入口，并显示“日历”入口。
- 手机 Web 包使用 `DUOYI_WEB_TARGET=mobile` 独立构建，保留移动端底部导航布局。
- 小组件页在电脑 Web 下不再展示添加桌面小组件按钮，改为显示月历预览和日历入口。
- 通用设置行统一最小行高和右侧操作居中，减少通知设置中图标、标题和副标题上下错位。
