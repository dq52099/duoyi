# v1.1.34

本版本修复 Android 覆盖安装仍提示“无法降级安装”的问题，并收紧强制更新策略。

## 修复

- Android `versionCode` 提升到 `140000`，高于此前 `v1.1.33` 的 `130105`，避免正式旧包覆盖安装时被系统误判为降级。
- 应用首次打开后立即检查管理员强制更新策略，命中后直接进入全屏强更页，不再依赖“我的 - 检查更新”手动触发。
- 强更页使用当前主题背景图和主题色，保留小屏滚动能力。
- 后端和客户端都会拦截明显指向旧版本 APK 的下载地址，优先使用发布通道中的正确安装包。
- 本地已下载 APK 的复用增加版本 URL 校验，避免旧缓存包被当成新版本安装。

## 版本

- Android `versionName`: `1.1.34`
- Android `versionCode`: `140000`
