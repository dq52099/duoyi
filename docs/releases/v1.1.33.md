# v1.1.33

本版本修复应用更新链路，确保新安装包不会被 Android 判定为降级安装。

## 修复

- Android `versionCode` 提升到 `130105`，高于 `v1.1.32` 的 `130104`。
- 修复强制更新策略：移动端接口返回强更时按实际新版本 / 新 build code 生效，旧版本配置不会误触发降级强更。
- 修复“我的”底部入口小红点：有新版本时会和未读通知一样提示。
- 支持已下载安装包离线安装：下载成功后记录本地 APK，断网、重开应用或授予安装权限后不再重复下载。

## 版本

- Android `versionName`: `1.1.33`
- Android `versionCode`: `130105`
- 签名配置保持不变。
