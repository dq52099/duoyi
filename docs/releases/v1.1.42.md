# v1.1.42

本版本修复登录后卡顿、成就重复弹出、提醒注册可靠性、已注册提醒明细和习惯多时间提醒问题。签名配置保持不变，正式 Release 仍由 GitHub Actions 使用既有 release keystore 构建。

## 修复

- 登录后账号资料刷新合并/节流，“我的”和资料页改为字段级监听，头像缓存不再随统计更新时间反复失效，降低登录态滑动卡顿。
- 登录或云同步恢复已解锁成就时，将既有成就写入已通知基线，避免“衣橱达人”等已读成就重复弹出。
- Android 原生全屏闹钟在定时注册前预创建状态通知和兜底通知渠道，不再依赖先点击“测试强提醒铃声”；全屏启动被系统拦截时保留高优先级兜底通知。
- 锁屏全屏闹钟调整为大时钟、可滚动布局，并更新标题、正文和按钮文案。
- “1 分钟后弹出测试”明确前台弹窗、后台/锁屏通知兜底的行为，弹窗出口不可用时自动降级为系统通知。
- 已注册提醒明细默认展示“生效中”，支持状态过滤和分页。
- 待办新建、编辑和“加入今日”默认等待提醒同步；修改提醒时旧提醒清理失败不再阻塞当前提醒重新注册。
- 习惯打卡提醒默认使用弹出框，隐藏提醒方式选择，支持多个每日提醒时间的展示、编辑、删除和逐条注册。
