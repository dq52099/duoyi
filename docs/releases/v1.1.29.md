# v1.1.29

本版本继续修复 v1.1.28 后仍存在的提醒、铃声试听、桌面小组件和卡顿问题。Android `versionCode` 提升到 `130101`，避免覆盖安装被系统识别为降级；签名配置保持不变。

## 修复与优化

- 修复立即发送通知、定时发送通知在替换旧通知前执行完整 cancel / 原生强提醒清理，导致普通通知 show / schedule 被清理失败阻断的问题。
- 铃声试听改为 Android 原生预览优先；当原生预览因媒体音量、初始化或设备限制失败时，自动回落到 Flutter 前台播放器路径，复用番茄专注可用的媒体播放链路。
- 修复桌面小组件快捷添加回调在部分 launcher 不返回 `EXTRA_APPWIDGET_ID` 时被误判失败的问题，改为确认但待验证状态并继续刷新 provider。
- 修复桌面小组件 RemoteViews 布局中使用裸 `View` 作为蒙层可能导致 launcher inflate 兼容性问题，统一改为受支持的 `TextView` 蒙层。
- 桌面小组件继续使用主题背景图 key 映射到 Android drawable，生成圆角 centerCrop 背景与轻量蒙层；主题切换后刷新小组件，不再只显示纯色。
- 通知栏今日进展同步增加进行中去重、失败退避和关闭后任务停止，避免同步失败进入重复刷新或重试造成 30 秒后持续卡顿。
- 延后并错开启动后的本地存储补载、平台服务、日历同步、小组件恢复和更新检查，减少进入 App 一段时间后的集中任务冲突。
- 黄历详情页修正宜 / 忌展示：每侧只保留一列竖排，最多展示 5 项，不显示更多、查看全部或省略号；详情页支持左右滑动切换日期。
- 继续收紧习惯页字体层级和卡片密度，修复通知设置页提醒方式 label 与选项拥挤问题。
