# v1.1.24

本版本聚焦日历/黄历交互、习惯页密度、主题小组件、通知设置、铃声试听和提醒可靠性修复。

## 修复与优化

- 日历页点击日期仅切换选中日期，不再自动打开黄历；黄历改为独立入口进入。
- 重构黄历详情页为全屏古典卡片布局，修复彭祖、冲煞和十二时辰区域裁切/遮挡问题。
- 收敛习惯页、本周概述、分组任务列表和通知设置页的字号、间距与卡片密度。
- 整理全局文字层级，移除页面局部过重字重回退。
- 首页/小组件主题数据同步真实背景图资源，带图主题不再退化为固定淡蓝色块。
- 优化主题背景与列表滚动绘制，降低滚动期间的图片和透明层成本。
- 修复通知设置页顶部安全区和滚动内容遮挡问题。
- 修复 Android 内置铃声试听，前台试听改为直接播放内置音频资源，不依赖通知权限或通知渠道。
- 改进铃声错误提示与日志，避免向用户展示原始异常堆栈。
- 保持任务提醒、闹钟和通知调度链路的权限/fallback/防重复逻辑，并补充回归测试。
