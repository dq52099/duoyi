# v1.1.31

本版本修复账号切换后的本机数据串号事故，并继续收口通知记录、提醒兜底、小组件和“我的”页稳定性。Android `versionCode` 提升到 `130103`，避免覆盖安装被系统识别为降级；签名配置保持不变。

## 修复与优化

- 退出账号和切换账号时清理本机账号数据，覆盖时光币、主题、习惯、头像缓存、提醒、小组件 payload 和账号本地目录，避免 admin 切 test 后继承旧账号状态。
- 通知记录默认保持未读，打开通知记录页不再自动清除未读状态，保留单条和批量手动已读。
- 通知设置增加注册通知入口，可查看已注册的普通提醒和闹钟提醒；立即发送、定时测试、弹出和闹钟路径增加失败反馈与兜底。
- 我的页头像区域使用固定 header 高度、稳定滚动状态和头像缓存 key，减少返回页面时的头像与顶部区域跳动。
- 弹出提醒增加系统通知兜底，后台或无可用前台上下文时仍保留可见提醒路径。
- 黄历 2026-06-28 的“宜”信息避免越框；AI 回顾默认折叠并支持展开；纪念日和生日卡片对齐倒数日展示方式。
- 小组件补齐 2x2、3x2、4x3 添加链路；待办未完成标记统一为 `o`；月历和待办小组件增加交互按钮，并优化应用内预览与桌面小组件布局。

## 发布说明

- Android `versionCode`: `130103`
