# v1.1.15

本次发布继续收口近期功能闭环、视觉统一和发布前回归修复。

## 重点修复

- Android versionCode 提升到 120015，避免安装时被系统判定为降级。
- 统一按钮、选中态、卡片、弹窗、表单和二级页面字号，减少突兀高亮和过大的跳转页字体。
- 修复倒数日新增或编辑后的保存反馈，保存后会关闭表单并显示成功提示，避免重复点击造成重复数据。
- 增加管理员创建用户 camelCase 布尔字段回归覆盖，确认 `isAdmin` 和 `isDisabled` 在未传 snake_case 字段时仍会生效。
- 保持现有 release 签名链路不变，继续由仓库 secrets 中的 keystore 配置完成正式签名。
