# v1.1.30

本版本继续收口“今日 / 万年历 / 黄历”视觉密度、习惯与任务列表滚动体验，以及 Android 通知注册失败后的可见诊断。Android `versionCode` 提升到 `130102`，避免覆盖安装被系统识别为降级；签名配置保持不变。

## 修复与优化

- 优化今日、万年历、黄历相关页面的卡片密度、文字层级、按钮对齐和详情页可读性，减少拥挤和遮挡。
- 收紧习惯热力图、今日打卡、任务列表和看板列表的滚动绘制范围，降低滑动卡顿和重复重绘。
- 通知栏今日任务进展只统计当天未完成任务，并在任务、习惯、目标变化后复用同一去重入口刷新常驻通知。
- 待办提醒、每日提醒和闹钟兜底链路在权限、插件或原生调度失败时保留可见失败原因，不再把失败误报为注册成功。
- Android 桌面小组件继续沿用主题背景、圆角和图标层级优化，保持与应用内视觉一致。

## 发布说明

- Android `versionName`: `1.1.30`
- Android `versionCode`: `130102`
- 签名配置保持不变，正式 Release 仍由 GitHub Actions 使用既有 release keystore 构建。
