# v1.1.19

本次发布继续收口主题切换、时光币同步、启动卡顿和小组件主题一致性。

## 重点修复

- 修复主题商店返回 `virtual_rewards` 但缺少直接 `coin_balance` 字段时，登录态时光币不刷新的问题。
- 云同步回写主题和时光币对象时，拒绝无 `updatedAt` 的旧远端对象覆盖本地新状态，降低主题自动回切风险。
- `/api/config` 启动刷新增加 in-flight 去重和短时缓存，减少冷启动重复请求。
- 冷启动后的账号刷新、提醒重放、小组件推送、更新检查和云同步进一步错峰，减少进入 App 后短时间内的主线程和平台通道压力。
- AI 每周回顾改为分层纯文本结构，避免长段落、Markdown 表格、emoji 和粗体。
- 习惯、日历页面和桌面小组件继续跟随当前主题与卡片皮肤。
- 保持 release 签名链路不变，继续使用正式 keystore，不改签名配置。
