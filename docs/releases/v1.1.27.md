# v1.1.27

本版本继续修复主题小组件、提醒铃声、通知触发链路、启动/滚动性能和日历/黄历体验问题，并将 Android `versionCode` 提升到 `130027`，避免覆盖安装时被系统识别为降级。

## 修复与优化

- 小组件在带背景图主题下使用对应主题图片，并保留轻量蒙层、圆角裁切和主题切换刷新。
- 铃声试听改为前台播放器播放内置 raw 资源，失败时只向用户展示简短提示，调试日志保留原始原因。
- 优化任务、习惯、每日提醒等通知标题和内容，保持强提醒与普通报告推送的渠道区分。
- 将启动后的通知入口、桌面小组件同步、平台服务和提醒恢复错峰执行，降低更新后首次进入和杀后台重进的阻塞。
- 优化主题背景图下采样、习惯页列表懒构建、日历日程预览和通知设置页滚动安全区。
- 调整习惯页字体层级、卡片密度、分组任务列表和热力图信息密度。
- 修复日历日期选择自动进入黄历的问题，黄历改为通过选中日期摘要卡进入详情。

## 发布说明

- Android `versionName`: `1.1.27`
- Android `versionCode`: `130027`
- 签名配置保持不变，正式 Release 仍由 GitHub Actions 使用既有 release keystore 构建。
