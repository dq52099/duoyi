# v1.1.26

本版本修复部分设备从近期测试包或内部构建安装正式 Release 时提示“不能降级”的问题。

## 修复与优化

- 将 Android `versionCode` 提升到 `130026`，确保正式包可以覆盖已安装的较高构建号，避免系统误判为降级安装。
- 保持应用签名链路不变，仍由 GitHub Actions 使用仓库既有 release keystore 构建正式 APK/AAB。
- 继承 v1.1.25 的通知小图标、铃声错误提示、黄历文本展示和通知渠道测试修复。

## 验证

- `dart format --set-exit-if-changed`
- `git diff --check`
- `flutter test test/core/app_version_static_test.dart test/services/reminder_ringtone_settings_test.dart test/services/notification_channel_version_static_test.dart`
- GitHub Actions release workflow 会继续执行完整 `flutter analyze`、`flutter test`、后端测试、Android release 构建和 Web release 构建。
