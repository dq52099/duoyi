# v1.1.36

本版本修复“我的”页顶部栏和统计卡片滚动回归，并继续收口近期通知、提醒和账号隔离问题。

## 修复

- “我的”页顶部标题栏恢复固定显示，只让资料、统计、AI 助手和菜单内容区滚动。
- “待办完成 / 连续打卡”等统计卡片改为稳定行列布局，避免 Flutter Web 滚动时局部闪烁或残留。
- 保留账号切换后的本机数据清理、待办滑动稳定性、通知记录和提醒调度修复。
- 收口全屏闹钟重复闪现路径，避免同一次提醒出现两段全屏入口。
