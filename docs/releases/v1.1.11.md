# v1.1.11

本次发布继续收口性能、通知、旧后端提示和反馈详情体验，重点修复高刷设备卡顿发热与普通页面直出接口诊断的问题。

## 重点修复

- 底部导航不再让已访问的离屏页面继续订阅 Provider，减少任务、日历、番茄钟等页面在后台反复重建。
- 首帧后启动任务错峰执行，优先处理初始 deep link 和分享文本，再异步刷新资料、提醒、通知栏和小组件。
- 更新后首次启动不再主动重建常驻通知栏入口，避免系统显示“马上有通知”的错觉。
- 云同步和共享空间后台请求遇到旧后端接口时，普通页面显示通用不可用文案，完整接口诊断保留给日志和手动诊断。
- 今日页共享空间只读任务禁止直接完成和删除，左滑详情会先收起操作按钮。
- 管理员反馈详情先打开列表摘要并显示加载态，慢接口或失败时不再表现为无法查看。
- 登录/注册主按钮、注册邮箱绑定区间距、我的页登录入口和头像 fallback 继续统一到当前紧凑风格。
- 反馈列表长内容、管理员回复和详情字段增加文本约束，减少正文区域被操作区挤压。
