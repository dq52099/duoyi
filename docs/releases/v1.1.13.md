# v1.1.13

本次发布修复安装降级提示、软件内更新版本回退和默认按钮风格问题。

## 重点修复

- Android versionCode 提升到 120013，避免从历史 split ABI 包切换到 universal APK 时被系统判定为降级安装。
- 软件内更新改为上报真实 Android build number，后端不再被旧 GitHub Release 拉回到 1.1.12。
- 后端默认当前版本同步到 1.1.13，并下发 120013 作为当前版本码。
- 默认按钮主题统一为更克制的尺寸、边框和选中态，减少突兀高饱和填充。
- TextButton、FilledButton、ElevatedButton、SegmentedButton 的前景色走对比度校验，避免选中后文字看不清。
