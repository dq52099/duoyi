# v1.1.14

本次发布集中交付近期任务闭环和发布前回归修复。

## 重点修复

- Android versionCode 提升到 120014，避免安装时被系统判定为降级。
- 默认铃声、提醒方式、通知栏任务进展、今日打卡、习惯结束/删除、倒数日、万年历、管理员分页和反馈详情等近期改动纳入发布。
- 修复管理员创建用户接口的 camelCase 布尔字段兼容问题，`isAdmin` 和 `isDisabled` 不再被默认值吞掉。
- 保持现有 release 签名链路不变，继续由仓库 secrets 中的 keystore 配置完成正式签名。
