# v1.1.20

本次发布继续收口启动卡顿、自习室异常循环、万年历视觉层级和桌面小组件主题一致性。

## 重点修复

- 冷启动后的账号刷新、提醒重放、通知栏刷新、小组件推送、日切处理、更新检查改为统一 idle 队列错峰执行，减少进入 App 后前几秒集中卡顿。
- 冷启动云同步和全量提醒重放进一步后移，避免刚打开 App 就和页面滑动、数据加载抢资源。
- 桌面小组件推送增加 in-flight 合并和更长防抖，避免多 Provider 连续刷新时重复重建日程摘要。
- 底部导航已访问页面保持挂载，日历、习惯、专注等页面来回切换不再反复整页重建。
- 自习室实时排行失败后的 UI 通知合并到 500ms，并增加远端失败冷却，WebSocket/接口异常时保持本地排行，减少重复报错和重绘。
- 万年历页面继续按小米日历结构弱化重边框，主页面保留月历、日期摘要和本月重点日期，详细黄历改为点击后查看。
- 习惯、日历和桌面小组件背景、按钮继续跟随当前主题，并保留圆角背景，不再被纯色背景覆盖。
