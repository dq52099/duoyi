# v1.1.25

本版本继续收尾日历、黄历、习惯、主题小组件和 Android 通知提醒链路，重点提升可用性、通知展示规范和发布包一致性。

## 修复与优化

- 补充 Android 单色通知小图标，任务提醒、铃声前台服务、地理围栏和专注音效前台服务不再复用启动器图标作为通知小图标。
- 统一 Flutter 本地通知与原生提醒通知的小图标配置，降低系统通知显示异常或被系统降级展示的风险。
- 收敛铃声试听失败时的用户提示，保留开发日志，避免普通用户看到平台通道或播放器原始异常。
- 加强通知渠道版本静态测试，确保任务、每日、习惯、报告和闹钟式提醒继续使用可区分的渠道策略。
- 补充铃声设置测试，覆盖平台通道失败时的简短提示。
- 优化黄历详情页日期信息换行策略，农历和干支信息优先完整展示，避免小屏下被省略或裁切。
