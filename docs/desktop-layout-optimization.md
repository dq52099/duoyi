# 桌面版布局优化方案

**版本**: v1.1.42  
**日期**: 2026-06-16  
**目标**: 全面重新设计桌面版，提升大屏体验

---

## 优化目标

1. ✅ 增大字体 - 适配大屏阅读
2. ✅ 增加间距 - 更宽松舒适
3. ✅ 优化侧边栏 - 独立设计
4. ✅ 提升对比度 - 更清晰
5. ✅ 重新设计排版 - 专属桌面体验

---

## 字体系统

### 当前字体（v1.1.41）
```
- 标题（greeting）: 22px
- 日期: bodySmall (约 12px)
- 卡片标题: 默认
- 正文: 默认
```

### 优化后字体（v1.1.42）
```
- 页面主标题: 28px (bold)
- 页面副标题（日期）: 15px
- 卡片大标题: 20px (medium)
- 卡片小标题: 16px
- 正文: 15px
- 辅助文字: 13px
```

---

## 间距系统

### 当前间距
```
- 外边距: 28px (左右), 22px (上)
- 卡片间距: 12-14px
- 内容列与侧边栏: 18px
- 侧边栏宽度: 340px
```

### 优化后间距
```
- 外边距: 40px (左右), 32px (上)
- 卡片间距: 20px
- 内容列与侧边栏: 28px
- 侧边栏宽度: 380px
- 卡片内边距: 20px (原 16px)
- 最大内容宽度: 1280px (原 1180px)
```

## 具体优化项

### 1. Header 区域
```dart
- 主标题: 28px, FontWeight.w600
- 日期: 15px, 包含农历、节气、星期
- 添加快速操作按钮行
- 高度: 120px
- 背景: 半透明卡片
```

### 2. 左侧边栏（新增）
```dart
- 宽度: 420px
- 内容:
  1. 精简月历（当月+高亮今日）
  2. 今日一言卡片
  3. 快捷功能（习惯、番茄钟、笔记）
- 固定定位，滚动独立
```

### 3. 主内容区
```dart
- 黄历卡片: 更大图标、更清晰宜忌
- 效率评分: 独立卡片，突出本周趋势
- 指标网格: 4列固定，更大数字显示
- 待办列表:
  - 按四象限分组
  - 每项高度增加到 56px
  - 更大的复选框和文字
```

### 4. 右侧边栏
```dart
- 宽度: 360px
- 目标: 进度条更粗，百分比更大
- 课程: 时间轴样式
- 纪念日: 倒计时更醒目
```

## 颜色优化

### 提升对比度
```dart
// 当前
onSurface: 默认
onSurfaceVariant: 默认

// 优化后
标题文字: onSurface.withAlpha(0.95) 
正文文字: onSurface.withAlpha(0.87)
辅助文字: onSurfaceVariant.withAlpha(0.70)
卡片背景: surface.withAlpha(0.88) → 0.92
边框: outlineVariant.withAlpha(0.36) → 0.45
```

---

## 响应式断点

```dart
// 三栏布局
>= 1400px: 左侧边栏 420px + 主内容 + 右侧边栏 360px

// 两栏布局
1100-1399px: 隐藏左侧边栏，保留右侧边栏

// 单栏布局（移动端）
< 1100px: 切换到当前移动端布局
```
