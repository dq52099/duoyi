# 多仪 v1.0.18 需求

## 必须修复

1. 月份/农历展示必须显示完整语义，例如 `丙午年（2026）三月初一`。
2. 日期选择控件统一美化，公历和农历明确区分。
3. 提醒默认时间取当前时间后的下一个半小时点，例如 17:28 → 17:30，17:35 → 18:00。
4. 白噪音增加种类、提高音量、延长循环长度，减少突然断开。
5. 通知设置要覆盖声音、震动、弹屏、强提醒测试和权限状态。
6. 日历支持手动选择日期，搜索放到右上角。
7. 新建任务优先级/四象限下拉框不得因键盘导致位置漂移。
8. 目标管理减少“保存本模块”类重复文案，开始/结束日期展示更紧凑。
9. 纪念日/生日/倒数允许在编辑时选择所有类型。
10. 今日日期卡弹出万年历式日期选择，不只给普通系统日历。
11. 子任务添加、勾选、删除后详情页展示必须立即同步。
12. 今日待办完成后不直接消失，应保留并允许取消完成。
13. 增加小部件菜单，展示可添加的小部件。
14. 通知设置增加每日提醒：开关、时间、今日任务、明日计划、逾期日程、重复周期、法定节假日暂停。
15. 热力图、统计、AI、小组件入口要继续向指尖时光体验靠拢。
16. 菜单栏至少提供默认入口配置，本轮保留后续自定义排序任务。
17. 后台管理继续沿统一控件风格优化。
18. AI 未启用时也要有可发现入口，说明配置后可用能力。
19. 增加/强化数据统计入口。
20. 专注设置支持任务名和更多时长预设。
21. 全项目不得残留系统 `showDatePicker` 与高粗体滥用。

- `flutter analyze` 通过。
- `flutter test --no-pub` 通过。

- GitHub Actions 构建成功并生成 `v1.0.18` Release APK。
