# 多仪 v1.0.16 需求

## 必须修复

1. 全局不再滥用粗体，普通正文、按钮、标签默认常规字重。
2. 白噪音最大音量明显提高。
3. 已完成任务可以取消完成。
4. 进度环不得遮挡 `100%`。
5. 今日页点击日期卡弹出日历。
6. admin/拥有者在私有清单和未加载共享空间时可以添加任务。
7. 新提醒默认时间为当前时间后的下一个半小时点，例如 11:20 默认 11:30，11:45 默认 12:00。
8. “我的/偏好”提供小组件说明入口。
9. 日历中间分割线降低对比度。
10. 新建任务先输入再选下拉框时，下拉框位置稳定。
11. 后台管理沿用统一视觉风格，减少粗体。
12. 习惯打卡提醒必须使用有声、震动、弹屏的强提醒。
13. AI 功能入口要可见，任务拆解和周回顾都能被发现。
14. 公历/农历选择区分组件；农历显示如 `丙午年（2026）三月初六`。
15. 通知设置页能验证普通通知、立即强提醒、定时强提醒和系统权限。

## 验收

- `flutter analyze` 通过。
- `flutter test --no-pub` 通过。
- `python3 -m unittest discover -s backend` 通过。
- GitHub Actions 构建成功并生成 Release APK。
