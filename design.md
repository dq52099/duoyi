# 多仪 v1.0.20 对齐修复设计

## 目标

本轮以“用户能立即感知、功能能闭环验收”为准：日期/时间/下拉控件统一，提醒链路可配置且可诊断，今日完成项不突然消失，白噪音连续且音量足够，AI、统计、小组件入口可发现。

## 设计原则

- 控件统一：日期、时间、弹框、下拉不再混用系统默认样式。
- 低字重：正文、按钮、卡片标题默认常规字重，不再用粗体堆视觉层级。
- 强提醒优先：习惯、任务闹钟走全屏、声音、震动通道；每日提醒可配置时间、内容和重复周期。
- 完成态可见：今日待办完成后保留在今日列表，可取消完成。
- 公历/农历分离：公历用月历，农历用年月日滚轮，展示干支年和公历年份。
- 白噪音以专注为目标：增加低频、夜雨、风扇等稳定音轨，降低短循环断点。
- 提醒弹屏闭环：任务/习惯强提醒点击后必须进入确认完成/打卡弹窗，而不是只停留在通知栏。
- 入口显性化：AI、小组件、统计、搜索放到用户能直接找到的位置。

## 主要改动

- 新增统一日期选择器 `AppDatePicker`，支持公历月历、农历滚轮、生日忽略年份语义。
- `AppDropdownField` 打开前统一收起键盘，解决新建任务优先级下拉框漂移。
- 偏好设置增加每日提醒：开关、时间、今日任务、明日计划、逾期日程、重复周期、节假日暂停。
- 今日页日期卡弹出万年历式日期选择，今日完成项保留展示。
- 日历页顶部支持手动选择日期，搜索移动到右上角。
- 目标编辑减少“保存本模块”噪音，开始/目标日期使用统一日期控件。
- 白噪音素材延长到 180 秒并提高增益，新增棕噪、夜雨、风扇、粉噪、低频溪流。
- 提醒默认时间改为“向后最近半小时点”，17:28 推荐 17:30，17:35 推荐 18:00。
- 任务提醒 payload 改为确认完成流程，闹钟通知带“完成任务/打开”操作。
- 日期选择器增加今天快捷入口，今日页日期卡选择后展示万年历详情。
- 热力图改为自适应尺寸、弱边框和 tooltip，减少拥挤。
- 我的页展示小部件菜单和 AI 配置状态，未启用 AI 时也能看到入口说明。

## 验收

- 扫描 `lib` 不应再有 `showDatePicker` 与 `FontWeight.w500+` 残留。
- `flutter analyze`、Flutter 单测、后端单测必须通过。
- 发布时 GitHub Release 必须包含 APK/AAB/Web 资产。
